    2. Install all dependencies:
       ```
       npm install react-router @tanstack/react-query @tanstack/react-table recharts framer-motion papaparse date-fns sonner xlsx zod react-hook-form @hookform/resolvers
       npm install -D @tailwindcss/vite @types/papaparse vitest @testing-library/react @testing-library/jest-dom jsdom @testing-library/user-event
       ```

    3. Initialize shadcn/ui: `npx shadcn@latest init` -- select style "new-york", CSS variables yes, base color neutral. Then add core components:
       ```
       npx shadcn@latest add button card dialog sheet alert-dialog input label select table badge skeleton sidebar dropdown-menu separator tooltip popover calendar tabs scroll-area avatar switch form toast
       ```

    4. Configure vite.config.ts:
       - Add @tailwindcss/vite plugin
       - Add proxy: `/api` -> `http://localhost:12345` (Encore backend)
       - Add path alias: `@` -> `./src`

    5. Create globals.css with Vercel Dashboard inspired theme:
       - Use `@import "tailwindcss"` (Tailwind v4 syntax)
       - Define CSS variables for light theme: near-white backgrounds (not pure white), near-black text (not pure black), green/cyan primary (hsl ~160, 84%, 39%)
       - Define `.dark` overrides: very dark backgrounds (not pure black, ~hsl(0,0%,4%)), light text
       - Ensure shadcn CSS variables are properly defined for both themes
       - Chinese-friendly font stack: `"PingFang SC", "Microsoft YaHei", system-ui, sans-serif`

    6. Create lib/theme/chart-theme.ts with CHART_COLORS constant (green, cyan, purple, orange, red, muted) as defined in RESEARCH.md.

    7. Create lib/utils.ts with cn() utility (clsx + tailwind-merge).

    NOTE: Use Tailwind CSS v4 syntax throughout -- no tailwind.config.js file. All customization via CSS @theme directive. Use `@import "tailwindcss"` not `@tailwind base/components/utilities`.
