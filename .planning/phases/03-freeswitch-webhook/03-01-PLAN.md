// callback/fsclient/interface.go
type FSClient interface {
    OriginateALeg(ctx context.Context, params OriginateParams) (uuid string, err error)
    OriginateBLegAndBridge(ctx context.Context, aUUID string, params OriginateParams) (bUUID string, err error)
    HangupCall(ctx context.Context, uuid string, cause string) error
    StartRecording(ctx context.Context, uuid string, callID string, leg string) error
    StopRecording(ctx context.Context, uuid string, callID string, leg string) error
    RegisterEventHandler(eventName string, handler func(CallEvent))
}

type OriginateParams struct {
    CallID      string
    Number      string
    CallerID    string
    GatewayIP   string
    GatewayPort int
    MaxDuration int
    Vars        map[string]string
}

type CallEvent struct {
    CallID      string
    UUID        string
    Leg         string    // "A" or "B"
    EventName   string    // "CHANNEL_ANSWER", "CHANNEL_BRIDGE", "CHANNEL_HANGUP"
    HangupCause string
    Timestamp   time.Time
}
```

  <action>
Create `callback/fsclient/esl.go` implementing the FSClient interface using eslgo (github.com/percipia/eslgo v1.5.0).

**ESLFSClient struct:**
- `conn *eslgo.Conn` -- the ESL connection
- `handlers map[string][]func(CallEvent)` -- event handlers registry
- `mu sync.RWMutex` -- protects handlers map
- `address string` -- ESL address (host:port)
- `password string` -- ESL password
- `onDisconnect func()` -- callback for Manager to detect connection loss

**NewESLFSClient(address, password string, onDisconnect func()) (*ESLFSClient, error):**
1. Create ESLFSClient struct
2. Call `eslgo.Dial(address, password, onDisconnect)` to establish inbound ESL connection
3. Subscribe to events via SendCommand: `event plain CHANNEL_ANSWER CHANNEL_BRIDGE CHANNEL_HANGUP`
4. Register global event listener via `conn.RegisterEventListener(eslgo.EventListenAll, client.dispatchEvent)`
5. Return client

**OriginateALeg(ctx, params):**
- Pre-generate A-leg UUID: `params.CallID + "-a"`
- Set origination variables: `origination_uuid`, `origination_caller_id_number`
- Call `conn.OriginateCall(ctx, true, aLeg, parkApp, vars)` where:
  - aLeg = `sofia/gateway/{GatewayIP}/{Number}`
  - parkApp = `&park()`
  - background=true for async originate
- Check response IsOk(), return UUID or error

**OriginateBLegAndBridge(ctx, aUUID, params):**
- Pre-generate B-leg UUID: `params.CallID + "-b"`
- Call `conn.OriginateCall(ctx, true, bLeg, bridgeApp, vars)` where:
  - bLeg = `sofia/gateway/{GatewayIP}/{Number}`
  - bridgeApp = `&bridge({aUUID})`
- Check response, return bUUID or error

**HangupCall(ctx, uuid, cause):**
- Call `conn.HangupCall(ctx, uuid, cause)`

**StartRecording(ctx, uuid, callID, leg):**
- Build recording path: `/var/lib/freeswitch/recordings/{callID}_{leg}.wav`
- Send raw API command via SendCommand: `uuid_record {uuid} start {path}`
- Create RawAPICommand helper that implements eslgo command.Command interface with `BuildMessage() string` returning `"api {command}"`

**StopRecording(ctx, uuid, callID, leg):**
- Build same path, send `uuid_record {uuid} stop {path}`

**RegisterEventHandler(eventName, handler):**
- Thread-safe append to handlers map using write lock

**dispatchEvent(event *eslgo.Event):** (private method)
- Extract event name via `event.GetName()`
- Only process CHANNEL_ANSWER, CHANNEL_BRIDGE, CHANNEL_HANGUP (skip others)
- Extract UUID from `event.GetHeader("Unique-ID")`
- Determine leg from UUID suffix: ends with "-b" = "B", otherwise "A"
- Extract callID by removing "-a"/"-b" suffix from UUID
- For CHANNEL_HANGUP: extract `event.GetHeader("Hangup-Cause")`
- Build CallEvent struct, dispatch to registered handlers (read lock)
- If no handlers registered for this event, log warning for orphan event (per CONTEXT.md: log warning with UUID + event details, don't crash)

**Compile-time interface check:**

```

**callback/fsclient/manager.go:**

`FSClientManager` struct:
- `instances [2]*managedInstance` -- exactly 2 (primary + standby)
- `mu sync.RWMutex`
- `activeIdx int` -- index of currently active instance (0=primary initially)
- `onFailoverCallback func(failedIdx int)` -- callback to finalize in-flight calls

`managedInstance` struct:
- `client FSClient` -- can be ESLFSClient or Mock (for testing)
- `address string`
- `password string`
- `healthy bool`
- `failCount int`
- `reconnecting bool`

`NewFSClientManager(primaryAddr, primaryPwd, standbyAddr, standbyPwd string, onFailover func(int)) *FSClientManager`:
- Create manager with two managedInstance entries, both initially unhealthy
- Do NOT connect in constructor -- use Connect() method

`Connect(ctx context.Context) error`:
- Attempt to connect both instances via NewESLFSClient
- Pass onDisconnect callback that calls manager.handleDisconnect(idx)
- Mark successfully connected instances as healthy
- Set activeIdx to 0 (primary) if primary connected, else 1
- Start health probe goroutines for each connected instance

`Pick() (FSClient, error)`:
- RLock, check instances[activeIdx].healthy first
- If unhealthy, check other instance
- If both unhealthy, return error "no healthy FreeSWITCH instance"

`startHealthProbe(idx int)`:
- Ticker every 10 seconds
- Send "status" command via ESL raw API
- On success: reset failCount to 0, mark healthy (but do NOT switch activeIdx back -- per CONTEXT.md: no auto-switchback after recovery)
- On failure: increment failCount. If failCount >= 3: mark unhealthy, if this was the active instance, switch activeIdx to other if healthy, call onFailoverCallback

`handleDisconnect(idx int)`:
- Immediately mark instance unhealthy (failCount = 3)
- If active instance, switch to standby
- Start reconnection goroutine with exponential backoff (1s, 2s, 4s... max 30s)
- On reconnect success: mark healthy, restart health probe

**For testing:** Accept FSClient interface in managedInstance (not just ESLFSClient) so tests can inject mocks.

`NewFSClientManagerWithClients(primary, standby FSClient, onFailover func(int)) *FSClientManager`:
- Test-only constructor that accepts pre-built FSClient instances
- Sets both healthy by default
