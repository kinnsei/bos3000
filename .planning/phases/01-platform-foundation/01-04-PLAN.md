Create `routing/migrations/1_create_gateways.up.sql`:
```sql
CREATE TABLE gateways (
    id BIGSERIAL PRIMARY KEY,
    name VARCHAR(100) NOT NULL,
    type VARCHAR(10) NOT NULL CHECK (type IN ('a_leg', 'b_leg')),
    sip_address VARCHAR(255) NOT NULL,
    weight INT NOT NULL DEFAULT 1,
    healthy BOOLEAN NOT NULL DEFAULT true,
    enabled BOOLEAN NOT NULL DEFAULT true,
    failover_gateway_id BIGINT REFERENCES gateways(id),
    carrier VARCHAR(100),
    health_check_failures INT NOT NULL DEFAULT 0,
    max_health_failures INT NOT NULL DEFAULT 3,
    last_health_check TIMESTAMPTZ,
    created_at TIMESTAMPTZ NOT NULL DEFAULT NOW(),
    updated_at TIMESTAMPTZ NOT NULL DEFAULT NOW()
);

CREATE TABLE gateway_prefixes (
    id BIGSERIAL PRIMARY KEY,
    gateway_id BIGINT NOT NULL REFERENCES gateways(id) ON DELETE CASCADE,
    prefix VARCHAR(10) NOT NULL,
    priority INT NOT NULL DEFAULT 0,
    UNIQUE(gateway_id, prefix)
);
CREATE INDEX idx_gateway_prefixes_prefix ON gateway_prefixes(prefix);
```

Create `routing/aleg.go`:
- `//encore:api private method=POST path=/routing/pick-a-leg` -- PickALeg:
  - No params needed (picks from pool)
  - Lock mutex, filter healthy+enabled gateways
  - Smooth weighted round-robin (nginx algorithm): for each healthy gw, currentWeight += weight; pick highest currentWeight; subtract totalWeight from winner
  - Return {GatewayID, Name, SIPAddress}
  - If no healthy gateways: NO_HEALTHY_GATEWAY error (503)

- Gateway admin CRUD (admin only):
  - `//encore:api auth method=POST path=/routing/gateways` -- CreateGateway
  - `//encore:api auth method=PUT path=/routing/gateways/:id` -- UpdateGateway
  - `//encore:api auth method=GET path=/routing/gateways` -- ListGateways (filter by type)
  - `//encore:api auth method=POST path=/routing/gateways/:id/toggle` -- ToggleGateway (enable/disable manual)

Create `routing/bleg.go`:
- `//encore:api private method=POST path=/routing/pick-b-leg` -- PickBLeg:
  - Accepts {CalledNumber string}
  - Extract prefix = CalledNumber[:3]
  - Query: SELECT g.* FROM gateways g JOIN gateway_prefixes gp ON g.id = gp.gateway_id WHERE gp.prefix = $1 AND g.enabled = true ORDER BY gp.priority ASC
  - Iterate results: pick first healthy gateway
  - If none healthy: follow failover_gateway_id chain (max 3 hops to prevent loops)
  - If still none: return PREFIX_NOT_FOUND (if no prefix match) or NO_HEALTHY_GATEWAY (if prefix matched but all down)
  - Return {GatewayID, Name, SIPAddress}

- Prefix CRUD (admin only):
  - `//encore:api auth method=POST path=/routing/gateways/:id/prefixes` -- AddPrefix
  - `//encore:api auth method=DELETE path=/routing/gateways/:id/prefixes/:prefix` -- RemovePrefix
  - `//encore:api auth method=GET path=/routing/gateways/:id/prefixes` -- ListPrefixes

Create `routing/did.go`:
- `//encore:api private method=POST path=/routing/select-did` -- SelectDID:
  - Accepts {UserID int64}
  - Step 1: SELECT number FROM did_numbers WHERE user_id = $1 AND status = 'available' ORDER BY RANDOM() LIMIT 1
  - Step 2 (if none): SELECT number FROM did_numbers WHERE user_id IS NULL AND status = 'available' ORDER BY RANDOM() LIMIT 1
  - If still none: return error "no DID numbers available"
  - Return {Number, DIDID}

- `//encore:api auth method=POST path=/routing/dids/import` -- ImportDIDs (admin only):
  - Accepts {Numbers []string, UserID *int64}
  - Bulk INSERT into did_numbers, skip duplicates (ON CONFLICT DO NOTHING)
  - Return {Imported, Skipped int}

- `//encore:api auth method=POST path=/routing/dids/:id/assign` -- AssignDID (admin only):

- `//encore:api auth method=POST path=/routing/dids/:id/unassign` -- UnassignDID (admin only):

- `//encore:api auth method=GET path=/routing/dids` -- ListDIDs (admin):

Create `routing/health.go`:
- Define a private endpoint for health check execution:
  `//encore:api private method=POST path=/routing/health-check` -- RunHealthCheck
  - Query all enabled gateways from DB
  - For each gateway: attempt TCP connection to SIP address port (net.DialTimeout, 5s timeout)
  - On success: reset health_check_failures to 0, set healthy=true
  - On failure: increment health_check_failures. If failures >= max_health_failures (default 3), set healthy=false
  - UPDATE gateway in DB, also update in-memory state (s.aLegGateways for A-leg gateways)
